Install

After you made all your changes in files, please do following on your FTP server and make it online.

FTP Upload:

  1. Open up your FTP manager and connect to your hosting
  2. Browse to required directory (Normally public_html)
  3. Upload the files inside vcard folder.

And then you can access your template on your main domain.

Font

When you open one of HTML files, you'll find this font inclution at top of the file. This is a main font including example with Google link.

    <!-- Fonts -->
    <link href='http://fonts.googleapis.com/css?family=Lato:100,300,400,700,900|Oswald:400,300,700' rel='stylesheet' type='text/css'>

Then you should set it on style.css file in a line like following

    css-selector {
      font-family: Oswald;
    }
    css-selector-2 {
      font-family: Lato;
    }

Logo & Branding

Easiest way to set your site logo is just replace images/logo.php file with yours.

But if you need to set different image on there, please just replace file name on header styling section of style.css file.

Less

This theme uses Less styling for development part. But if you want to use this dynamic styling for your site, you should edit following files.

    /ls
      - blog.less
      - comments.less
      - content.less
      - elements.less
      - footer.less
      - header.less
      - helpers.less
      - page.less
      - portfolio.less
      - responsive.less
      - style.less
      - variables.less
      - widgets.less

And finally we need to call less compiler on your heading section.

    <!-- Less Compiler -->
    <link rel="stylesheet/less" type="text/css" href="less/style.less" />
    <script>
    less = {
      env: "development",
      async: false,
      fileAsync: false,
      poll: 1000,
      functions: {},
      dumpLineNumbers: "comments",
      relativeUrls: false,
      rootpath: "./"
    };
    </script>
    <script src="js/less.min.js"></script>
